Informativa sulla Privacy – NexuOne

Ultimo aggiornamento: 13/01/2026

Titolare/Developer: FrancescoTech

Contatti: francescotech95@gmail.com

NexuOne (“App”) include funzionalità come scanner PDF, gestione note e strumenti di produttività, oltre a funzionalità di trasferimento file verso server FTP configurati dall’utente.

1. Dati trattati

L’App è progettata per funzionare principalmente in locale sul dispositivo. In generale, non raccogliamo né conserviamo su nostri server i contenuti creati dall’utente (es. PDF scansionati, note, file).

  • Contenuti dell’utente: immagini acquisite per scansioni, PDF generati, note e file selezionati dall’utente.
  • Dati di configurazione: preferenze dell’app e impostazioni (es. connessioni FTP) salvate localmente.

2. Permessi e loro utilizzo

  • Fotocamera (CAMERA): utilizzata per acquisire immagini/documenti e creare scansioni PDF.
  • Accesso a file/foto/media: usato per selezionare, salvare, aprire e condividere file scelti dall’utente.
  • Internet: necessario per connessioni a server FTP configurati dall’utente.

3. Trasferimenti verso server FTP scelti dall’utente

Se l’utente utilizza le funzioni FTP, l’App si connette a un server inserito e scelto dall’utente per trasferire file. I dati trasferiti vengono inviati al server FTP scelto dall’utente, non ai nostri server.

4. Credenziali FTP

Se l’utente salva una connessione FTP, l’App può memorizzare localmente sul dispositivo i dati di connessione (ad esempio host, porta, nome utente e password) per permettere connessioni rapide. L’utente può eliminare le connessioni salvate in qualsiasi momento dalle funzionalità dell’App oppure disinstallando l’App.

5. Condivisione dei dati

Non vendiamo dati personali e non condividiamo dati con terze parti per finalità pubblicitarie.

I dati possono essere condivisi solo:

  • su azione esplicita dell’utente (es. “Condividi” un PDF tramite altre app, invio tramite FTP);
  • per obblighi di legge.

6. Servizi di terze parti

L’App può utilizzare componenti di sistema necessari al funzionamento su Android e alla distribuzione tramite Google Play. Se in futuro venissero integrati servizi di analytics o crash reporting, questa informativa verrà aggiornata.

7. Conservazione dei dati

I contenuti e le impostazioni sono conservati localmente sul dispositivo finché l’utente non li elimina o disinstalla l’App.

8. Sicurezza

Adottiamo misure ragionevoli per proteggere i dati nell’App. Si consiglia di proteggere il dispositivo con PIN/biometria.

9. Privacy dei minori

L’App non è destinata specificamente a minori di 13 anni e non raccoglie consapevolmente dati personali di minori.

10. Diritti dell’utente

  • eliminare i contenuti dall’App o dal dispositivo;
  • revocare i permessi dalle impostazioni del dispositivo;
  • contattarci a: francescotech95@gmail.com

11. Modifiche
